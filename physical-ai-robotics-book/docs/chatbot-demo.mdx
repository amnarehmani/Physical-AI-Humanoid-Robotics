# Book Q&A Chatbot Demo

This is a demonstration of the integrated chatbot for querying the book content.

## Using the Chatbot

To use the chatbot:

1. Select text from this page or elsewhere on the site
2. Type your question in the input box below
3. Press Enter or click "Send"
4. The bot will respond with information from the book

## Features

- Global book querying via `/query` endpoint
- Contextual selection-based querying via `/query/selection` endpoint
- Loading indicators
- Error handling
- Chunk ID display for traceability
